# Glance — Design Inspector

A Figma-style design inspector for any webpage. Inspect spacing, typography, colors, and layout without leaving your browser.

## Install

Chrome doesn't require the Web Store for local extensions. Here's how to load it in a few steps:

### 1. Download the extension

Click the green **Code** button on this page → **Download ZIP**, then unzip the folder somewhere you'll remember (e.g. your Desktop).

### 2. Open Chrome Extensions

Go to `chrome://extensions` in your address bar.

### 3. Enable Developer Mode

Toggle **Developer mode** on in the top-right corner.

### 4. Load the extension

Click **Load unpacked**, then select the unzipped `glance` folder.

That's it — the Glance icon will appear in your Chrome toolbar.

## Usage

- Click the **Glance icon** in the toolbar, or press **Cmd+Shift+U** (Mac) / **Ctrl+Shift+U** (Windows/Linux) to toggle the inspector.
- Hover over any element to inspect its styles.
- Click an element to pin the inspector to it.

## Updates

To update to a newer version, download the latest ZIP, replace the old folder with the new one, then go to `chrome://extensions` and click the refresh icon on the Glance card.
